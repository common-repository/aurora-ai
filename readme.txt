Aurora AI plugin allows you to add an AI-powered chatbot to your website.

== Description ==

Aurora AI integrates an AI-powered chatbot into your website. This plugin provides an easy way for website owners to manage interactions with users via AI.

**This plugin relies on an external service provided by Aurora AI (https://www.itsaurora.ai).**

The plugin connects to the Aurora AI platform to embed the chatbot on your website. By using this plugin, you agree to Aurora AI's [Terms of Service](https://www.itsaurora.ai/terms) and [Privacy Policy](https://www.itsaurora.ai/privacy).

== Installation ==

1. Upload the `aurora-ai` folder to the `/wp-content/plugins/` directory.
2. Activate the plugin through the 'Plugins' menu in WordPress.
3. Configure the plugin by adding your chatbot ID in the settings page under "Aurora AI".

== Frequently Asked Questions ==

= Where can I get the Chatbot ID? =

You can get your chatbot ID from the [Aurora AI dashboard](https://www.itsaurora.ai/login) after configuring your chatbot.
